Reading Assistant
Консольное Java-приложение, которое принимает ссылку на статью, автоматически загружает её содержимое, выполняет анализ (саммари, ключевые слова, основные идеи) и сохраняет результат в виде PDF-файла.
Проект построен на Spring Boot и использует собственный конвейер обработки текста. PDF создаётся через Apache PDFBox.

Возможности
Загрузка статьи по URL
Извлечение текста
Генерация саммари и ключевых тезисов
Формирование PDF в директории output
Работа из консоли и возможность создания .exe через jpackage

Пример использования
java -jar reading-assistant.jar https://example.com/article


После обработки в папке output появится файл вида:
2025-02-15_article_summary.pdf
с текстом анализа.
